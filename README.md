# SQL Learning Project (MySQL)

This repository documents my hands-on practice while learning **SQL and relational databases** through the **SQL Tutorial for Beginners (Complete Course using MySQL)** by **CodeWithHarry** on YouTube.

## Project Overview
In this project, I designed a **Student Course Management Database** and practiced writing SQL queries to understand how relational databases work in real-world systems.

The database includes four tables:
- Students
- Courses
- Instructors
- Enrollments

These tables demonstrate **relationships between entities** and how SQL joins connect data across multiple tables.

## Topics Practiced
- Database creation and table design
- Data insertion using `INSERT`
- Data retrieval using `SELECT`
- Joins (`INNER JOIN`, `LEFT JOIN`)
- Aggregate functions (`COUNT`, `SUM`, `AVG`, `MAX`)
- Grouping and filtering (`GROUP BY`, `HAVING`)
- Subqueries and derived tables
- Analytical queries on relational data

## Files Included
- Database design diagram
- Table creation queries
- Data insertion queries
- 25 basic SQL queries
- Advanced SQL queries using aggregation and subqueries
